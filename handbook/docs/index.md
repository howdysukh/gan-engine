# Gan Engine

## Mathematical Market Monitoring Engine

Welcome to the official documentation for Gan Engine.

Gan Engine is an open-source Progressive Web Application that helps investors monitor mathematically significant price levels using the Gann Square of Nine methodology.

Unlike traditional trading platforms, Gan Engine does **not** predict future prices. Instead, it continuously evaluates live market data against predefined mathematical levels and notifies users when those levels are reached.

---

## Features

- Google Authentication
- Guest Sessions
- Live Market Monitoring
- Gann Square of Nine Engine
- Watchlists
- Browser Notifications
- Progressive Web App
- MongoDB Atlas
- Express.js Backend
