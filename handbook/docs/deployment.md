# Deployment Guide

### Gan Engine

Version 0.5

</div>

---

# Requirements

- Node.js
- MongoDB Atlas
- Google OAuth Credentials
- GitHub Repository
- Render Account

---

# Clone Repository

```bash
git clone https://github.com/yourusername/gan-engine.git
```

---

# Install

```bash
npm install
```

---

# Configure Environment

```env
PORT=3000

MONGO_URI=

JWT_SECRET=

GOOGLE_CLIENT_ID=

GOOGLE_CLIENT_SECRET=

GOOGLE_CALLBACK=
```

---

# Start

```bash
npm start
```

---

# Deploy to Render

1. Create Web Service
2. Connect GitHub
3. Select Repository
4. Build Command

```
npm install
```

5. Start Command

```
npm start
```

6. Add Environment Variables

7. Deploy

---

# Verify

- Homepage loads
- Google Login works
- Guest Login works
- Watchlists load
- Alerts load
